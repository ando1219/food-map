# 🍜 大学城美食地图

> 帮助大学城高校联盟的同学快速发现周边美食，解决「今天吃什么」的世纪难题。

**在线访问** → [ando1219.github.io/food-map](https://ando1219.github.io/food-map)

---

## 功能介绍

- **按学校筛选** — 覆盖 A/B/C/D 四所高校，选择你的学校只看周边餐厅
- **按校区定位** — 支持主校区、北校区等多校区细分，精准缩小范围
- **输入当前位置** — 选择宿舍、教学楼、图书馆等具体位置，自动按步行距离排序
- **预算 & 菜系筛选** — 按人均消费和菜系快速过滤
- **随机推荐** — 选择困难？一键随机推荐一家
- **餐厅详情** — 查看推荐菜品、同学评价、步行时间

## 项目结构

```
food-map/
└── index.html    # 全部代码（HTML + CSS + JS 合并在一个文件）
```

## 本地运行

直接双击 `index.html` 用浏览器打开即可，无需安装任何依赖。

## 添加新餐厅

打开 `index.html`，找到 `RESTAURANTS` 数组，按照以下格式添加一条数据：

```js
{
  id: 11,                          // 唯一编号，自增
  name: "餐厅名称",
  cuisine: "菜系",                  // 快餐 / 面馆 / 韩餐 / 轻食 / 烧烤 等
  schools: ["A", "B"],             // 覆盖哪些学校
  budget: 15,                      // 人均消费（元）
  rating: 4.5,                     // 评分（0-5）
  hot: true,                       // 是否热门
  x: 1.0, y: 1.5,                  // 坐标（用于计算距离，参考已有数据估算）
  img: "https://图片链接",
  dishes: [
    { n: "招牌菜名", p: "¥15" }
  ],
  reviews: [
    "同学评价一",
    "同学评价二"
  ]
}
```

改完后执行以下命令更新线上页面：

```bash
git add index.html
git commit -m "新增餐厅：xxx"
git push
```
